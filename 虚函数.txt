0、前提：
	理解虚函数的前提必须要先理解多态
	
1、多态：	
	定义：指相同的对象收到不同的消息（静态多态）或者不同的对象收到相同的消息（动态多态）时产生不同的动作
	静态多态：
		类中有两个同名函数（重载）
			int calcArea(int width);
			int calcArea(int width, int height);
		成员函数名相同，收到不同的消息时会产生不同的动作，即函数里面的操作不同。
		当我们实例化后
			Rect rect;
			rect.calcArea(10);
			rect.calcArea(10, 20);
			在编译阶段就确定了执行函数，这种就被称为静态多态
	动态多态：
		例如：当需要计算一个圆与一个长方形的面积，对象不同（圆、长方形），收到的指令都为计算面积，产生的动作（计算过程）不同
				这种就被称为动态多态
		类中有两个成员函数
			int circle(int r);
			int calcArea(int width, int height);
		注意：动态多态的前提必须以封装与继承为基础。

2、继承：这里不探讨继承的语法。主要看实例。
	情况一：两个子类继承父类的计算面积函数，通过父类指针调用时，都是调用到父类的计算函数。我们要的是操作到子类的计算函数。
	情况二：同上，但是在父类的计算面积函数前加virtual，则调用时操作的就是子类的计算函数
	所以，虚函数的目的是子类继承父类后，通过父类指针操作子类的函数（同名函数），具体参考代码。
