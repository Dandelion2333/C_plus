一、构造函数
	目的：初始化类的成员变量
	写法：函数名与类名相同，不用返回值，不用定义类型
		Cdate::Cdate()
		{
			_year = 1900;
		}
	重载：支持，也就是支持传入参数，改变成员变量的值
	重载的写法：在构造函数的基础上加变量传入
		Cdate::Cdate(int year)
		{
			_year = year;

			cout << _year << "重载完成" << endl;
		}

二、析构函数
	目的：类执行完后做清理变量内存工作，释放内存
	写法：函数名在类名之前加~，不用返回值，不用定义类型
		Cdate::~Cdate()
		{
			cout << _year << "析构函数执行完成" << endl;
		}
	重载：不支持
	顺序：当有多个类被析构时，最后被调用的类先被析构。可参考栈的存储结构。
	注意：对象被销毁前调用析构函数，由于不同的对象作用域不同，所以调用析构函数的时机也不同
		1、全局对象：当程序执行完成后被销毁
		2、自动对象：当离开对象的作用域后被销毁
		3、动态对象：当执行delete运算符后被销毁
